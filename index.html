What is Mannat?

A Universal DevOps & Cloud System.

Mannat is a cloud-agnostic automation platform and lifecycle orchestration engine. It unifies the entire software lifecycle, Build, Test, Deployment, Runtime, and Cloud Orchestration, into a single, intelligent system.
